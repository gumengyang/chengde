## 前端运行

```bash
# 进入项目目录
cd ruoyi-wms-lite-vue

# 安装依赖
npm install --registry=https://registry.npmmirror.com

# 启动服务
npm run dev

# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:80
```
